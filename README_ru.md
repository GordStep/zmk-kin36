# ZMK прошивка для Kin36, Sweep36, Sweep Squared
[English version](README.md)

ZMK прошивка для клавиатур Kin36, Sweep36 (щит ZMK Cradio, но с 36 клавишами), Sweep Squared, Swoop36 и других клавиатур формата 3x5 с полной бездиодной матрицей и стандартным BLE-микроконтроллером, совместимым с ProMicro (nice!nano, ProMicro NRF52840, TENSTAR ROBOT NRF52840 и другие).

## Быстрый старт

### Прошивка новой версии

1. Сделайте форк этого репозитория
2. (опционально) Обновите раскладку и файл конфигурации или использовать этот [сайт](https://nickcoutsos.github.io/keymap-editor/)
3. Дождитесь сборки прошивки в GitHub Actions, затем скачайте и распакуйте zip-архив
4. Подключите левую или правую половину клавиатуры к компьютеру с помощью USB-кабеля
5. Переведите левую или правую половину в режим загрузчика. Сделать это можно следующими способами:
   1. дважды нажав кнопку сброса
   2. быстро замкнув контакты кнопки сброса дважды (если кнопка не работает)
   3. быстро замкнув контакты GND и RST контроллера дважды (не замыкайте контакты VCC 3.3V или RAW 5V!)
6. Скопируйте прошивку на соответствующую половину клавиатуры (при обновлении раскладки или названия клавиатуры прошивать правую половину не требуется)
   1. файл с пометкой `left` предназначен для левой половины
   2. файл с пометкой `right` — для правой половины
   3. файл с пометкой `reset` используется для любой половины для сброса информации о Bluetooth-соединениях (применяйте, когда левая и правая половины не могут соединиться, чтобы забыть сопряжённые устройства)

## Символы

- Слой по умолчанию и слой Windows — это слои для ввода символов.
- На обоих слоях клавиши-модификаторы `CTRL`, `OPTION (ALT)`, `COMMAND (WINDOWS)` расположены на тех же позициях, что и символьные клавиши под безымянным, средним и указательным пальцами обеих рук. Удерживайте клавишу для активации модификатора или нажмите её кратко, чтобы ввести символ.

## Цифры и навигация

- Удерживайте правый Tab для перехода на правый слой — здесь можно вводить цифры или управлять навигацией.
- Отпустите правый Tab, чтобы вернуться на слой по умолчанию или Windows.
- Специальные клавиши:
  - `&bootloader` — переводит правую половину в режим загрузчика, чтобы скопировать на неё новую прошивку.

![правый слой](images/right-layer.jpg)

## Знаки пунктуации

- Удерживайте левый Tab для перехода на левый слой — здесь можно вводить знаки пунктуации.
- Отпустите левый Tab, чтобы вернуться на слой по умолчанию или Windows.
- Специальные клавиши:
  - `&default_report` — выводит информацию о заряде батареи
  - `&bootloader` — переводит левую половину в режим загрузчика, чтобы скопировать на неё новую прошивку
  - `5` — переключает на слой мыши

![левый слой](images/left-layer.jpg)

## Функциональные клавиши

- Удерживайте одновременно левый и правый Tab для перехода на третий слой — здесь доступны функциональные клавиши.
  - `BT_SEL_#` — выбор слота Bluetooth-профиля, к которому вы подключаетесь или хотите подключиться
  - `BT_CLR` — очистка соединения в выбранном слоте для повторного сопряжения устройства с этим слотом
  - `OUT_TOG` — переключение между USB и Bluetooth, позволяя использовать до 6 подключений (5 Bluetooth + 1 USB)
  - `&tog 1` — включение/выключение слоя Windows (переключение между слоем по умолчанию и слоем Windows)
  - `&studio_unlock` — разблокировка клавиатуры для использования [ZMK Studio](https://zmk.dev/docs/features/studio#keymap-changes) для переназначения клавиш
  - `&soft_off` — переход в режим программного выключения (аналогично глубокому сну после часа бездействия, но пробуждение возможно только специальными клавишами, которые настроены на левый большой палец: `shift`)
- Отпустите клавиши Tab, чтобы вернуться на слой по умолчанию или Windows.

![третий слой](images/tri-layer.jpg)

## Мышь

- Удерживайте левый Tab (для входа в левый слой) и нажмите клавишу пробела, чтобы перейти на слой мыши
- Нажмите `p` или `q`, чтобы выйти из слоя мыши
- `MB4` — кнопка «Назад», `MB5` — кнопка «Вперёд»
